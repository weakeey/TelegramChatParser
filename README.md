# TelegramChatParser

Простой парсер активных пользователей Telegram чата.

Необходимые библиотеки для запуска скрипта: telethon, pandas, colorama

Измените api_id и api_hash для работы скрипта.

После запуска укажите @username или ChatID и количество сообщений для парса - скрипт отберет уникальных и запишет в Excel таблицу.
